PartCallbacks

Description: This example demonstrates how a custom application is added to NX. It defines menubar applications with callbacks for both the C++, .Net, Java, and Python and adds the applications to the start menu.

Files Required for example

This table represents the various demos for registering part callback for every available reason via the different languages.
Language Application Files Men Files Note
C++ MenuBarCppApp.cpp & MenuBarCppApp.h MenuBarCppApp.men & MenuBarCppAppButton.men Both the files must be compiled together
C# MenuBarCSharpApp.cs MenuBarCSharpApp.men & MenuBarCSharpAppButton.men
JAVA MenuBarJavaApp.java MenuBarJavaApp.men & MenuBarJavaAppButton.men
Python MenuBarPythonApp.py MenuBarPythonApp.men & MenuBarPythonAppButton.men

How to Compile the Files: The detailed description on how to compile the files in various languages is available in MenuBar_CompileLink_README.html There is no steps needed to build Python.

NOTE:It is not required to build all of the examples, just build the languages that interest you (if applicable).

How to Run the Example

    [A] Create Directory structure

    Create an application root.

    Make a directory called MenuBarApplicationRoot, and make two subdirectories in this directory. With one called application and the other startup. And then place the following files in the directories.
    • MenuBarApplicationRoot (directory)
      • application (directory)
        • MenuBarCppApp.dll (This file is obtained from the build steps For C++ binary. You will want to rename the dll to have this name)
        • MenuBarCppApp.men
        • MenuBarCSharpApp.dll (This file is obtained from the build steps For C Sharp binary. You will want to rename the dll to have this name)
        • MenuBarCSharpApp.men
        • MenuBarJavaApp.jar
        • MenuBarJavaApp.men
        • MenuBarPythonApp.py
        • MenuBarPythonApp.men
      • startup (directory)
        • MenuBarCppAppButton.men
        • MenuBarCSharpAppButton.men
        • MenuBarJavaAppButton.men
        • MenuBarPythonAppButton.men

    Set the varaible UGII_USER_DIR pointing to the full path of directory MenuBarApplicationRoot in the NX command prompt.

    [B] Execute the Example

      NOTE: In the workflow below replace LANG with the language you want to run. For Example CPP for C++.

    1. Start NX
    2. Create New Part or Open an Existing Part. (The Start Menu won't appear until a part is open)
    3. To launch Sample LANG application, go to Start->All Applications-> Sample LANG Application. This will create "Sample LANG" menu next to Help.
    4. Click on the different buttons under the menu "Sample LANG" to verify the callbacks associated with the button names.
