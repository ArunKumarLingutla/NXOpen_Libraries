How to Build C++ examples
 -------------------------------

Windows
  1. Using Visual Studio - create a new project using the NX Open AppWizard. This is described in the Automation -> NXOpen -> Open for C/C++ -> Open C Reference Guide -> Overview -> Setting up your system. When setting the Application Settings, select "Create an Internal application".
  2. Remove all the existing files from project.
  3. Add the C++ implementation an header file to Source Files and Header Files respectively by selecting the Project pulldown and select "Add Existing Item...". Select the file using the popup window.
  4. Build the internal image by going to the Build menu and selecting either "Build Solution" or "Rebuild Solution".

This will create an executable in the Debug directory.

How to Build C# examples
 ----------------------------------
  1. Open Microsoft Visual Studio .Net.
  2. Create a new C# project using the NX Open .Net Template. This is described in the NX Open Programmers Guide -> Wizard Setup.
    Also information under Creating NX Open Automation -> Applications will be helpful. When setting the Application Settings, select "Create an Internal application" and select NXOpen.UI API.
    Select to activate the application Explicitly and to unload when NX terminates. Then pick Finish.
  3. Add the cs_part_callbacks.cs to the project by selecting the Project pulldown and select "Add Existing Item...". Select the file using the popup window.
  4. Add a reference to Microsoft.VisualBasic. This is typically done from the Solution Explorer and Right Mouse Button on References, and add References.
  5. Build the internal image by going to the Build menu and selecting either "Build" or "Rebuild".
This will create an .Net executable.
